# 早上早起就是为了吃

|  起床日期  | 起床时间 | 天气 |  起床心情  |
| :--------: | :------: | :--: | :--------: |
| 2020-05-18 |  05:42   | 多云 | just so so |

周日跑了一天，玩上七点多就睡了，一口气睡到早上五点多，起床打个哈欠伸个懒腰，感觉还不错。

烧水，牛肉解冻，靠墙站立 10min，然后起身去厨房。

豆奶，牛肉，煎蛋。这早餐吃一辈子都不会腻。一顿早餐约合人民币 28 元(●'◡'●)，得找个性价比高的方式才行，不然会被吃穷。

本来昨晚要准备入职材料的，结果昨天睡得太香了。

去卫生间自拍了半个小时的自拍，然后出发去新公司。

希望今天入职顺利。
